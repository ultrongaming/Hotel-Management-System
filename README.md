# Hotel Management System 🏨

A Hotel Management System developed using Python that helps manage hotel operations such as customer details, room management, and booking-related activities through a simple interface.

## Features

- Customer registration and management
- Room management system
- Hotel booking functionality
- Store and retrieve customer details
- Database integration for storing records
- User-friendly interface with images and visual assets

## Project Structure

```
Hotel Management System/
│
├── hotel.py              # Main application file
├── customer.py           # Customer management module
├── room.py               # Room management module
├── hotel.db              # Database file
│
├── hotel images/         # Images and UI resources
│
└── tempCodeRunnerFile.py
```

## Technologies Used

- Python
- SQLite Database
- Tkinter (GUI)
- File Handling

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/hotel-management-system.git
```

2. Open the project folder:

```bash
cd hotel-management-system
```

3. Install required dependencies (if needed):

```bash
pip install -r requirements.txt
```

4. Run the project:

```bash
python hotel.py
```

## Database

This project uses SQLite database:

- `hotel.db`

The database stores:
- Customer details
- Room information
- Booking records
